# TcPkgMgr-PowerShellScript

A numeric menu driven PowerShell script for the TwinCAT Package Manager. This PowerShell script provides an interactive console interface for managing TwinCAT packages. This code was created entirely by Claude. If you find any issues with it or would like to see a feature added, please submit a GitHub issue or Pull Request.


This can be run from the PowerShell ISE or from a command prompt using "powershell -ExecutionPolicy Bypass -File "C:\path\to\your\script.ps1"


==============================================================
  TwinCAT Package Manager Console   |   Main menu
  Mode: LIVE
==============================================================

   1. Packages & workloads
   2. Sources (feeds)
   3. Configuration
   4. Tasks (automation)
   5. Search files in local packages
   6. Search files in feed packages
   7. Toggle dry-run mode
   8. Run a raw tcpkg command
